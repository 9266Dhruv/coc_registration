# ⚔️ Clash of Clans — Clan Dashboard

A beautiful clan management dashboard for Clash of Clans tournaments, member registration, and announcements.

## 🚀 Features
- **Dashboard**: Tournament stats, registrations, member overview
- **Tournament Registration**: Sign up for clan war leagues
- **Admin Panel**: Create tournaments, post updates, manage members
- **Auto CWL**: Automatically generates CWL tournaments on the 26th of each month
- **Responsive Design**: Works on mobile, tablet, and desktop
